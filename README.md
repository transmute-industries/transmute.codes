# RaptorQ Compressed QR Code Data URI Transmission

Needs a better name.

This is how it works:

```mermaid
flowchart LR
  File
  DataURI
  RaptorQ
  GZIP
  QRCodes

  File --> DataURI
  DataURI --> RaptorQ
  RaptorQ --> GZIP
  GZIP --> QRCodes
```
